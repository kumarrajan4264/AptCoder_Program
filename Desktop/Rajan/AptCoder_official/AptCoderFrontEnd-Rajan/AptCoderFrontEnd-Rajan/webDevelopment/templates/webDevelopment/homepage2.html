Teaching kids the Apt Coder way

When it comes to learning we believe in holistic approach- in which we
learn everything by doing it practically at the same time keeping the
learning process friendly to the learner. There are various ways of doing the
same thing, and by being smart we can save lots of energy for things that
are yet undone.
A thousand miles journey starts from first step.
You understand the code and you can play with the program.
